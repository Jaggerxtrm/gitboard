# Intro
Questo documento descrive metodologie specifiche per un uso più efficiente degli agenti derivante da idee prese da repo pre-esistenti.
Sono principalmente una raccolta di skill e mcp a disposizione degli agenti. Gli agenti spawnati non possono essere lasciati "liberi" di organizzarsi come vogliono, ma guidati per svolgere al meglio il loro lavoro.

# Repo
https://github.com/frankbria/ralph-claude-code.git - sistema di loop che fa continuare gli agenti a svolgere il lavoro fino a che non è terminato. Utile per evitare eventuali blocchi.
https://github.com/gsd-build/get-shit-done - development organizzato, suddivide task in blocchi, si assicura che vengano terminate (PRD>tasks).
https://github.com/coollabsio/jean?tab=readme-ov-file - UI grafica. Forse meglio di cli?
https://github.com/affaan-m/everything-claude-code - molto da prendere qui.
